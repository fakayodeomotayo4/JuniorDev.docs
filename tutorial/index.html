CSS Flexbox

The complete beginner's guide to flexible layouts. No prior CSS layout knowledge required.

What is Flexbox?

Flexbox is a CSS layout method that lets you arrange items in a row or column. It handles
spacing, alignment, and ordering — all the things that used to require hacks with floats and
positioning.

NOTE

Flexbox works in one dimension at a time — either a row or a column. If you need two-dimensional
layouts, look into CSS Grid.

Getting Started

To use Flexbox, you apply display: flex to a parent container. All direct children become
"flex items."

.container {
display: flex;
}

/* That's it. Your items are now in a row. */ Direction

Direction

By default, flex items lay out in a row (left to right). You can change this with flex-
direction.

.container {
display: flex;
flex-direction: column; /* Stack items vertically */
}

/* Options: row | row-reverse | column | column-reverse */

Alignment

This is where Flexbox really shines. Two properties handle most alignment needs:

.container {
display: flex;
justify-content: center; /* Horizontal alignment */
align-items: center; /* Vertical alignment */
}

/* justify-content options:
flex-start | flex-end | center |
space-between | space-around | space-evenly */

/* align-items options:
flex-start | flex-end | center |
stretch | baseline */

Pro Tip

To perfectly center something both horizontally and vertically, use justify-content: center and align-items: center together. This is the easiest centering method in CSS.

Flex Grow, Shrink & Basis

These three properties control how items resize:

.item {
flex-grow: 1; /* How much to grow relative to siblings */
flex-shrink: 0; /* How much to shrink (0 = don't shrink) */
flex-basis: 200px; /* Starting size before growing/shrinking */
}

/* Shorthand: */
.item {
flex: 1 0 200px; /* grow shrink basis */
}

Common Patterns

Here are layouts you'll use every day:

Navigation Bar

nav {
display: flex;
justify-content: space-between;
align-items: center;
}

Card Grid

.cards {
display: flex;
flex-wrap: wrap;
gap: 1rem;
}

.card {
flex: 1 1 300px; /* Grow, shrink, min 300px */
}

Sticky Footer

body {
display: flex;
flex-direction: column;
min-height: 100vh;
}

main {
flex: 1; /* Takes all available space */
}
