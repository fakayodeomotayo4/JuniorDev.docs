JavaScript Arrays

map, filter, reduce — the holy trinity. Learn them once, use them forever.

Arrays 101

An array is an ordered list of values. It can hold numbers, strings, objects — anything.

const fruits = ["apple", "banana", "cherry"];
const numbers = [1, 2, 3, 4, 5];
const mixed = ["hello", 42, true, { name: "Ada" }];

// Access by index (starts at 0)
fruits[0]; // "apple"
fruits[2]; // "cherry"

// Length
fruits.length; // 3

Adding & Removing

const arr = [1, 2, 3];

// Add to end
arr.push(4); // [1, 2, 3, 4]

// Remove from end
arr.pop(); // [1, 2, 3]

// Add to beginning
arr.unshift(0); // [0, 1, 2, 3]

// Remove from beginning
arr.shift(); // [1, 2, 3]

// Remove/insert at any position
arr.splice(1, 1); // removes 1 item at index 1 → [1, 3]

.map() — Transform Every Item

Creates a new array by running a function on every item. The original array is untouched.

const numbers = [1, 2, 3, 4];

const doubled = numbers.map(n => n * 2);
// [2, 4, 6, 8]

const names = ["ada", "grace", "margaret"];
const capitalized = names.map(name =>
name.charAt(0).toUpperCase() + name.slice(1)
);
// ["Ada", "Grace", "Margaret"]

Note

The second argument to .reduce() is the initial value. Always provide it to avoid unexpected behavior with empty arrays.

.find() & .includes()

const numbers = [1, 2, 3, 4, 5];

// Find first match
numbers.find(n => n > 3); // 4

// Check if value exists
numbers.includes(3); // true
numbers.includes(99); // false

// Find index of first match
numbers.findIndex(n => n > 3); // 3

Chaining Methods

The real power comes from combining these methods:

const students = [
{ name: "Ada", grade: 92 },
{ name: "Grace", grade: 78 },
{ name: "Alan", grade: 95 },
{ name: "Linus", grade: 64 },
];

// Get names of students who passed (grade >= 70), sorted
const honor = students
.filter(s => s.grade >= 70)
.map(s => s.name)
.sort();
// ["Ada", "Alan", "Grace"]
