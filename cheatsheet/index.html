Flexbox Cheat Sheet

Every Flexbox property in one place. Search or scroll.

Container Properties

PROPERTY VALUES DESCRIPTION
display flex | inline-flex Defines a flex container
flex-direction row | column | row-reverse | column-reverse Sets the main axis direction
flex-wrap nowrap | wrap | wrap-reverse Controls whether items wrap to new
lines
justify-
content 		flex-start | flex-end | center | space-between | space-around | space-
evenly		 Aligns items along the main axis
align-items stretch | flex-start | flex-end | center | baseline Aligns items along the cross axis
align-content stretch | flex-start | flex-end | center | space-between | space-around Aligns rows when wrapping
gap < length > Sets spacing between flex items


Items Properties

PROPERTY VALUES DESCRIPTION
flex-grow < number > How much an item grows relative to siblings
flex-shrink < number > How much an item shrinks relative to siblings
flex-basis < length > | auto The initial size before growing or shrinking
flex < grow > < shrink > < basis > Shorthand for grow, shrink, and basis
align-self auto | flex-start | flex-end | center | stretch | baseline Overrides align-items for one item
order < integer > Changes visual order without changing DOM
