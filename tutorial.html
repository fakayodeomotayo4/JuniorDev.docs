CSS Flexbox

The complete beginner's guide to flexible layouts. No prior CSS layout knowledge required.

What is Flexbox?

Flexbox is a CSS layout method that lets you arrange items in a row or column. It handles spacing, alignment, and ordering — all the things that used to require hacks with floats and positioning.

NOTE

Flexbox works in one dimension at a time — either a row or a column. If you need two-dimensional layouts, look into CSS Grid.
